# Machine Learning

## Përshkrimi i detyrës

Ky projekt synon të zgjidhë sfidat e orarizimit në universitete teknika të avancuara të Machine Learning. Ne jemi fokusuar në analizën dhe përpunimin e dataseteve të propozuara gjatë viteve 2020-2023, të cilat përfshijnë të dhëna të orarizimit në universitete.
